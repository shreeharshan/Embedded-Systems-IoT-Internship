# Task 2 – Automatic Light Detection System

## Objective

To design and simulate an Automatic Light Detection System using ESP32 and an LDR sensor.

## Components Used

- ESP32 DevKit V1
- LDR Sensor Module
- LED
- 220Ω Resistor
- Breadboard
- Jumper Wires

## Working

The ESP32 reads the LDR sensor's analog output. When the ambient light level falls below a predefined threshold, the LED turns ON. When the light level is above the threshold, the LED turns OFF.
