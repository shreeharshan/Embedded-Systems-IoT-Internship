# Embedded Systems & IoT Internship

This repository contains the tasks completed during my **Embedded Systems & IoT Internship** at **MainCrafts Technology**.

## Repository Structure

- 📁 Task-1 – Smart Temperature and Humidity Monitoring System
- 📁 Task-2 – Automatic Light Detection System using ESP32 & LDR
- 📁 Task-3 – IoT-Based Smart Automatic Lighting System using ESP32, LDR & Blynk Cloud
- 📁 Task-4

## Technologies Used

- ESP32
- Arduino (C++)
- Wokwi Simulator
- Blynk IoT Platform
- DHT22 Sensor
- LDR Sensor
- LCD1602 I2C
- Wi-Fi Communication
- Git & GitHub
