## Task 1: Smart Temperature and Humidity Monitoring System

### Project Overview
This project uses an ESP32 microcontroller and a DHT22 sensor to monitor temperature and humidity in real time. The measured values are displayed on a 16×2 I2C LCD. Whenever the temperature exceeds **35°C** or the humidity exceeds **80%**, an LED turns ON and a buzzer generates a repeating alert.

### Components Used
- ESP32 DevKit V4
- DHT22 Temperature & Humidity Sensor
- LCD1602 I2C Display
- LED
- Buzzer
- 220Ω Resistor
- Breadboard and Jumper Wires

### Features
- Real-time temperature monitoring
- Real-time humidity monitoring
- LCD display output
- LED warning indicator
- Repeating buzzer alert
- Simulated using Wokwi

### Project Files
